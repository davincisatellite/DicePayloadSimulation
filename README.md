# Dice Payload Simulator

[screenshot](assets/screenshot.png)

A minimal web-based dice simulator for payload testing and demos.

Features
- Simple in-browser dice rolling and visualization.

Usage
- Open [index.html](index.html) in your browser or run a local server:

```bash
python -m http.server
```

Files
- [index.html](index.html) — UI
- [script.js](script.js) — simulation logic
- [style.css](style.css) — styles
- [assets/](assets/) — images and assets
